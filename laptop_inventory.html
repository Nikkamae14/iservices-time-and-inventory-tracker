📁 OneDrive Setup

Important: Follow these steps to sync data across all devices:
Step 1: Create a folder in OneDrive (e.g., "LaptopInventory")

Step 2: Upload the laptop_inventory.html file to that folder

Step 3: Right-click folder → Share → "Edit" permissions

Step 4: A laptopInventoryData.json file will auto-create on first save

✅ All laptops will now see the SAME data in real-time!
